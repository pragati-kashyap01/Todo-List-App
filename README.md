# Todo List App

## Project Description
This is a simple Todo List Web Application created using HTML, CSS, and JavaScript.  
It allows users to add tasks dynamically and display them on the screen.

This project is built to practice basic DOM manipulation and event handling in JavaScript.

## Features
- Add new tasks
- Display tasks instantly
- Clean and simple UI
- Beginner friendly project

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User enters a task in the input field.
2. Clicks the "Add Task" button.
3. A new task element is created using JavaScript.
4. The task is displayed inside the task list section.
